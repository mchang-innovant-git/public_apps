# YYZ ⇄ HKG Fare Tracker (Winter 2026/27)

Daily price tracker for three Toronto–Hong Kong trips (Cathay Pacific nonstop vs Air Canada via YVR, Economy + Premium Economy).

- `index.html` — the dashboard (GitHub Pages)
- `data/history.json` — full daily price history, one snapshot per day

Data is collected daily from Google Flights and pushed automatically. Prices are CAD, round-trip, per person, taxes included.
